Protein AI Pipeline

Research implementation project exploring multimodal protein function inference using ESM-3 embeddings, BLIP-2 vision-language generation, and GO similarity evaluation.

GitHub WorkstreamOpen Poster

Role

  • Research implementation
  • ESM-3
  • BLIP-2
  • Evaluation
  • Python

Source repo: Cao-Labs/lutesAI2025.

Contribution scope: implementation and experimentation in the public ChrisB folder/workstream, not ownership of the entire lab repository.

Problem

Protein function inference requires connecting sequence-derived representations, visualized similarity structures, and interpretable semantic evaluation in a way reviewers can understand.

Process

The implementation work maps protein sequences into ESM-3 embeddings, generates similarity-matrix visuals, runs BLIP-2 caption workflows, and evaluates outputs with GO similarity scoring.

Outcome

The project demonstrates applied AI research workflow design: embedding generation, visual representation, caption generation, and evaluation scripts connected into a reproducible pipeline.
