Opioid Prescribing Risk Analysis

A healthcare data project using public prescribing datasets to explore opioid prescribing patterns and model risk-related signals.

GitHub: ChristopherDSBarker

Problem

Public prescribing data is large, technical, and hard to interpret quickly. The project frames that data into a risk-analysis story a reviewer can understand.

Process

Cleaned and summarized prescribing data, compared geographic and provider-level patterns, built model outputs, and translated findings into charts and poster visuals.

Outcome

The result is a polished analysis with visible model interpretation and presentation-ready visuals.
