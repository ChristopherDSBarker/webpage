Context

This project defines the visual identity now used across the portfolio. The emblem functions as a persistent brand anchor for the navbar, favicon, hero watermark, and site-wide authorship system.

Design System

The system centers on a layered lotus/seal structure with gold, red, and navy hierarchy, star rhythm, Korean center symbolism, and serif luxury tone. The website uses a cropped authentic emblem for UI scale while preserving the full artwork as the master brand artifact.

Outcome

The branding package turns a decorative mark into a repeatable identity system: detailed master artwork for presentation, cropped emblem for UI, and consistent placement across the portfolio navigation layer.

Project Package

The source folder is the canonical branding archive. The deployable website exposes the complete design story: the final emblem system, master board composition showing the full visual system, typeface study documenting typography exploration, sketch iterations revealing design process, and the authentic navbar crop for UI scale.
