# Image-Aware Caption Generator (Simple Version)

A single-file mini-app that generates social media captions based on an uploaded image and a selected vibe.  
Designed for classroom demonstration, minimal, AI-assisted, and fully deployable.

---

## Features

- Upload an image (jpg/png)
- Choose **vibe**: Funny, Aesthetic, Casual, Confident, Emotional, Minimal
- Generate **5 AI-powered captions**
- Regenerate captions if needed

---

## Requirements

- Python 3.10+
- Streamlit
- OpenAI Python SDK
- OpenAI API key
