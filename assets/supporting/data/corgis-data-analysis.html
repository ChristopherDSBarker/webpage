1. Summary of the CORGIS Paper¶

CORGIS (Collection Of Really Great and Interesting Datasets) is an open source project designed for accesible/introductory computing and data science education. Demographic is non-computer science majors.¶

Two learning methods, Situated Learning Thoery and MUSIC(Model of Academic Motivation); according to the PDF, students learn better when working in authenitc, real-world contexts, and Students engage more when activities feel useful, interesting, empowering, and supported. The challenge is finding datasets for use.¶

Corgis has 40+ datasets for use. Covering wide range of fields: health, politics, history, education, environment, etc. Auto-generated materials used datasets files in CORGIS follows: Python, Java, Racket Libraries, JSON, CSV, SQL, Documentation and metadata. Vizualition tools for histograms, scatterplots, line charts and bar charts; no programming needed. Shows structure of dataset and support data mapping plus navigation.¶

Non-majors have Computation thinking course. Explore graphing activities. Programming pactice problems; mapping complex data structures. A final project where students pick dataset and perform mini data analysis. Evaluation: datasets to complex, to many fields and hiearchy is bad. Numeric and string data dominate. Real world data helps motivation. Keep students computing, highlight programming skills connect career benefits.¶

def predict_popularity(celebrity=False, funny=False, animals=False):
    """
    Predicts whether an ad will be 'Popular' or 'Less Popular'
    based on patterns found in the Super Bowl dataset.
    """

    # Weighted scoring based on real dataset patterns
    score = 0

    if celebrity:
        score += 2.5   # strongest correlation with high views/likes
    
    if funny:
        score += 1.5   # second strongest
    
    if animals:
        score += 0.5   # small 

    # Threshold from stats: mean popularity ≈ 1.4M views
    if score >= 2.5:
        return "Predicted: Popular Ad (high views/likes)"
    else:
        return "Predicted: Less Popular Ad"
